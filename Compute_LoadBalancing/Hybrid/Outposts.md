Hybrid cloud: business that keep an on-premises infra alongside a loud infra

# 1. OVERVIEW

AWS Outposts = Fully managed service (dịch vụ được quản lý hoàn toàn)

- Extend AWS infrastructure, services, APIs, và tools đến customer premises
- Cung cấp phần cứng được quản lý bởi AWS tại chỗ
- Đám mây riêng tại chỗ

Bản chất: extends aws experience to on-premises, là 1 server-rack.

# 2. Đặc điểm

- Full managed by aws (aws-managed infras tại custom site)
- Hybrid cloud experienced

# 3. Service trên outposts

├─ EC2 (Amazon EC2) ├─ EBS (Amazon EBS) ├─ ECS (Amazon ECS - container-based) ├─ EKS (Amazon EKS - container-based) ├─ RDS (Amazon RDS - database) ├─ EMR (Amazon EMR - analytics) └─ S3 (Amazon S3 - storage)

# 4. Kết luận

AWS Outposts là fully managed service mở rộng AWS infrastructure, services, APIs, tools đến customer premises (tại chỗ), cung cấp phần cứng AWS được quản lý hoàn toàn, cho phép chạy applications tại chỗ với low-latency, xử lý data local, và tuân thủ data residency.
