# AWS directory services

1. AWS managed microsoft AD: trust relationship between onprem and aws (1 way, 2 way), backup, MFA.
2. AD connector: support MFA.
3. Simple AD: No support MFA.

Note: Active Directory Replication architecture

Connection between onprem and aws using DX (direct connect) or VPN

Simple AD is the cheapest option and basic AD compatible.

AWS managed microsoft AD is the most expensive option but highest AD compatible.
